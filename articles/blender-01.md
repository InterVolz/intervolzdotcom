## Hot Hotkeys

| key          | purpose          | bonus               |
| ------------ | ---------------- | ------------------- |
| Left Mouse   | Orbit            |                     |
| Right Mouse  | Deselect?        |                     |
| Middle Mouse | Pan              |                     |
| Middle Mouse | Pan              |                     |
| Tab          | Edit Mode Toggle |                     |
| G            | Move             | ZXY to lock axis    |
| S            | Scale            | ZXY to lock axis    |
| Alt-G        | Reset location   |                     |
| Alt-S        | Reset scale      |                     |
| 1            | Select Vertex    |                     |
| 2            | Select Edge      |                     |
| 3            | Select Face      |                     |
| Shift-A      | Create Object    |                     |
| NUM1         | View X           |                     |
| NUM2         | View Y           |                     |
| NUM3         | View Z           |                     |
| Ctrl - R     | Edge Loops       | mousewheel for more |
| I            | Inset            |                     |
